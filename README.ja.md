# Previous River

Obsidian用のプラグインで、ノートの `previous` プロパティやバックリンクに基づいてノート間を移動できます。

## 機能

### 前のノートに移動
現在のノートのフロントマターにある `previous` プロパティでリンクされたノートに移動します。

### 次のノートに移動
現在のノートにバックリンクを持ち、かつその `previous` プロパティが現在のノートを指しているノートに移動します。  
候補が複数ある場合は、選択用のモーダルが表示されます。

### 最初のノートに移動
`previous` プロパティのチェーンをたどり、シーケンス内の最初のノートに移動します。

### 最後のノートに移動
バックリンクをたどり、シーケンス内の最後のノートに移動します。  
候補が複数ある場合は、選択用のモーダルが表示されます。

## ホットキーの例

- **前のノートに移動**: `Alt+,`
- **次のノートに移動**: `Alt+.`
- **最初のノートに移動**: `Alt+Shift+,`
- **最後のノートに移動**: `Alt+Shift+.`
